UI Mockup: Library Room Booking System

This project is a mockup for a potential application for an improvement on UF's library room booking system. Through observation, our group found many study rooms left unoccupied even when there was no availability through the LibCal system. Therefore, we decided that an application using QR codes and geofencing could be a solution to the problem.

The "Quick Book" feature allows users to be able to reserve currently open rooms in one click". "Reserve a Room" gives the user an interface to work with that it much more straight forward than the current LibCal system. The "Check Room Status" feature has the user scan a QR code (each individual study room has its own unique QR code) in order to check the reservtion status of the room, and possibly request a room swap.

Room swapping is a huge factor of this project - rather than unused rooms just going unused, we can optimize the space utilization by haivng users in need request to take the room from people who reserved a room but simply do not need it anymore. Our entire system solves the issue of ambiguity with reservations, and frustrations of never being able to find an open room.

This interactive mockup was created in Adobe XD.
